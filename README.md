# KAAPAV Phase Classification — VALOR NLP Layer

NLP group project. Supervised classification of sales call segments into 
six KAAPAV phases (Kontakt, Analyse, Angebot, Pruefung, Abschluss, Verstaerkung).

## Repository Contents

- `01_kaapav_finetune_colab.ipynb` — Fine-tuning pipeline: BERT (bert-base-german-cased), 
  zero-shot baseline (XLM-R), Viterbi order-smoothing, evaluation on gold set
- `02_clustering_eda_colab.ipynb` — Exploratory clustering analysis justifying the 
  supervised classification framing
- `data/synthetic_transcripts.jsonl` — Synthetic training transcripts (448 labelled segments)
- `data/gold_eval.jsonl` — Held-out evaluation set (448 labelled segments)

## How to Run

1. Open `01_kaapav_finetune_colab.ipynb` in Google Colab
2. Upload `synthetic_transcripts.jsonl` and `gold_eval.jsonl` when prompted
3. Run all cells (Runtime → Run all)
4. GPU runtime recommended (T4 or above)

## Requirements

All dependencies are installed in the notebook setup cell:
transformers, datasets, evaluate, accelerate, scikit-learn, seaborn, sentence-transformers, umap-learn

## Note on Evaluation Data

The gold evaluation set is synthetic, generated by the same script as the training data. 
Results reflect in-distribution performance. Generalization to real-world calls is a 
stated limitation and direction for future work.
